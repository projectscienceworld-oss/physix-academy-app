# **App Name**: PhysixAcademy

## Core Features:

- Topic-Driven Video Hub: A curated library of video lectures organized by physics chapters (Mechanics, Optics, etc.) with search, filters, and a 'Mark as Watched' progress tracker.
- Interactive Simulations Gallery: Integration with PhET and oPhysics via iframes, featuring custom 'What to observe' guides for immersive conceptual learning.
- Live Class & Countdown: A centralized dashboard for live session access (Zoom/Meet integration) with an active countdown timer for the next lecture and recording archives.
- Weekly Physics Routine: A color-coded weekly timetable with PDF export functionality, allowing students to toggle between schedule lists and visual calendars.
- Tiered Assessment Engine: A four-level difficulty quiz system with real-time feedback, performance badges, and automated LaTeX equation rendering using KaTeX.
- Numerical AI Assistant: An AI-powered tool to help students break down complex numerical problems into logical steps with clear, reasoning-based explanations.
- LaTeX Problem Repository: Structured problem sets with collapsible step-by-step solutions, highlight boxes for key formulas, and marking for student revision.

## Style Guidelines:

- Primary color: Electric Cobalt (#2E79F3) for a professional and energetic academic focus.
- Background color: Deep Navy Night (#0D1117), providing a modern, dark academic aesthetic that emphasizes text clarity.
- Accent color: Celestial Azure (#00C6FF), used for highlighted formulas, timers, and active status indicators.
- Headline font: 'Space Grotesk' for a technical, scientific appearance. Body font: 'Inter' for high legibility in dense academic text.
- Formula styling: KaTeX/LaTeX integration for all equations to ensure industry-standard scientific typesetting.
- Utilize Lucide React for consistent, thin-stroke scientific icons that represent physics branches like atoms, circuits, and magnets.
- Responsive layout with a persistent dark sidebar for desktop and a quick-access bottom navigation bar for mobile students.
- Smooth transitions between course chapters and subtle height expansions for collapsible step-by-step solutions.
